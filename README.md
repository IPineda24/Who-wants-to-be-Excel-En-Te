# ¿Quién quiere ser Millonario? – Excel Edition

## Requisitos
- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Logo

Coloca tu archivo `logo.png` en la carpeta `/public/`.

## Build para producción

```bash
npm run build
npm start
```

## Funciones del juego

- **50:50** – Elimina dos respuestas incorrectas
- **Saltar** – Omite la pregunta actual (se puede reintegrar)
- **Eliminar** – Borra la pregunta permanentemente
- **Reintegrar** – Devuelve las preguntas saltadas al pool
- **Reiniciar** – Empieza desde el principio
